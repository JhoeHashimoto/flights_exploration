# Exploração de Voos  
Esta é uma ferramenta para explorar dados de voos, oferecendo visualizações e análises para entendê-los melhor.  

## Funcionalidades  
- Visualização de rotas de voos  
- Análise de dados de voos  
- Comparação de preços  

## Como Usar  
1. Clone o repositório.  
2. Instale as dependências.  
3. Execute o aplicativo.
