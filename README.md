# Flights Exploration Project

## Overview
The Flights Exploration Project is designed to allow users to explore and analyze flight data efficiently. This project aggregates flight information from various sources, providing insights into flight patterns, delays, and other relevant metrics. 

## Features
- **Data Aggregation**: Collect flight data from multiple APIs and sources.
- **Data Visualization**: Display data trends through charts and graphs.
- **User-Friendly Interface**: Intuitive interface for users to interact with data easily.
- **Analytics**: Perform various analyses on flight data, such as delay causes and frequency of flights.

## Getting Started
To get started with the Flights Exploration Project, follow these steps:
1. Clone the repository: `git clone https://github.com/JhoeHashimoto/flights_exploration.git`
2. Install the required dependencies: `pip install -r requirements.txt`
3. Run the application: `python app.py`
